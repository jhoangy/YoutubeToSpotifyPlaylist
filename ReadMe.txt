  Autmate Spotify with Python

Extract music videos from your YouTube playlist and adds them to a Spotify playlist

  Getting Started

For this project, you will need Python installed on your system.

  Prerequisites

1. YouTube Data API credentials (named as `client_secret.json`). These must be placed in the creds/ directory.
2. Spotify username. This can be found on the Spotify Account Overview.
3. Spotify Web API OAuth Token. This can be generated from:
https://developer.spotify.com/console/put-current-user-saved-tracks/?ids=

  Installing

pip install -r requirements.txt

  Running the app

python run.py
